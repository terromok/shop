# shop
Реализовано:
+Авторизация
+Товары можно покупать на странице shop/catalog Туда ведут все основные ссылки в верхнем меню. При нажатии на кнопку купить, в верхнем правом углу, возле значка корзины, высвечивается количество товаров в корзине. Правда, при обновлении страницы снова исчезает. 
+Пользователь в своей корзине может увеличивать или уменьшать количество определенного товара на одну единицу. Или сразу удалить товар из крзины. Данные выводятся асинхронно.
+Администратор может редактировать товар.
